Про меня

Меня зовут Виталий

38 лет, проживаю в г. Луганске.

Осваивать профессию Front-End Developer, начал с января 2021г.
Первоначально промониторил в интернете, как войти в IT, приобрел первые книги: "HTML и CSS3" (Эд Титтел, Крис Минник), "Большая книга CSS" (Дэвид Макфарланд), "Создаем динамические веб-сайты с помощью PHP, MySQL, JavaScript, CSS и HTML" Robin Nixon. Изучая данный материал, параллельно смотрел видео-контент от опытных верстальщиков и разработчиков, которые теорию подкрепляли практикой, таким образом, постепенно повышая свой практический опыт.

Для дальнейшего развития во Front-end, необходимо было структурировать материал обучения. В марте 2021г. приобрел два курса "WEB-разработчик - 2021" и "Полный курс по JavaScript + React - с нуля до результата" от Ивана Петриченко на платформе Udemy. Данные курсы упорядочили изучаемые технологии для Front-End, а также позволили закрепить основной базовый фундамент и принципы web разработки.

Основной стек: React, React + Redux, JS, HTML, CSS, SCSS или SASS, БЭМ и т.д.

Цель: стать высококвалифицированным специалистом в данном направлении, максимум через 3 года, с момента трудоутсройства, вырасти в Team Lead.

Личный план роста:
- уверенный junior: до 1 года,
- middle: от 1 до 2 лет,
- senior: от 2 - 3 лет.

Пожелания: расти и развиваться вместе с Kомпанией, внутри команды единомышленников.

Личные и деловые качества: работоспособность, ответственность, дисциплинированность, нацеленность на результат, адекватность, внимательность, порядочность, сосредоточенность, высокая степень самоорганизации, работа в команде, объективная оценка своих возможностей для решения поставленных задач, осваивание новых технологий и инструментов.

Навыки

Что я использую в работе

HTML5

Создает каркас сайта или приложения, а пятая версия позволяет создавать более SEO-оптимизированную структуру продукта

Использую: БЭМ, семантическую верстку и знаком с доступностью.

CSS3

Язык стилей позволяет создавать абсолютно любой внешний вид сайта или приложения.

Использую: Flexbox, Grid, Float, RWD, препроцессоры Sass/Scss, Bootstrap, CSS анимации

Java Script

Язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое.

ECMAScript (ES6+)

Jquery

Библиотека Jquery - это набор функций JavaScript, по взаимодействию JavaScript и HTML.

Использование: редко

React

JavaScript-библиотека для разработки пользовательского интерфейса.

Использую: Create React App, любые стили CSS, SCSS или Styled Component, методы жизненного цикла для КК или hooks для ФК, Portals, React Rooter v.5-v.6, React Transition Group, React Icons, Formik, Yup, Helper, React devTools и т.д.

Redux

Паттерн и библиотека для управления и обновления состояния приложения с использованием событий, называемых "действиями".

Использую: для соединения React + Redux - функцию connect или hooks, Redux devtools, store enhancer, middleware, Redux-thunk, Redux Toolkit.
